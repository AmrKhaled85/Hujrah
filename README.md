# рҹ“– About

**ШӯЩҸШ¬ШұШ© (Hojrah)** is a modern web platform developed as a Graduation Project during the **Digital Egypt Pioneers Initiative (DEPI)**.

The platform helps university students find trusted accommodations while enabling property owners to advertise their apartments in a secure environment.

The platform also provides an Admin Dashboard for managing users, properties, payments, approvals, and platform activities.

# рҹ’° Business Model

The platform generates revenue through a **2.5% commission** from the rental amount.

Example:

Rent = **6000 EGP**

Platform Commission = **150 EGP**

Students and property owners can unlock each other's contact information only after payment approval.
